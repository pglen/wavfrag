WavFrag Voice Recognition, KeyStrokes

  One can send arbitrary ASCII characters to the currently active window. The letters A-Z, the numbers from 0 to 9 and the punctuation marks are sent without any modifications to the currently active application. In addition, the following special sequences are recognized:
 
VK_RETURN   
The return key.
VK_SHIFT  

The shift key. This is a toggle key. Sending it the first time is equivalent to pressing the shift key, sending it the second time is equivalent to release the shift key.  The user is responsible to maintain the send count to get the expected results.
VK_CONTROL  
     
The control key. This is a toggle key. Sending it the first time is equivalent to pressing the control key, sending it the second time is equivalent to the release of the control key.  The user is responsible to maintain the send count to get the expected results.
VK_MENU   

The alt key. This is a toggle key. Sending it the first time is equivalent to pressing the alt key, sending it the second time is equivalent to the release of the alt key.  The user is responsible to maintain the send count to get the expected results.
VK_BACK  

The backspace key

To send a special key just embed it into the character sequence you are sending. For example to send shift down arrow, enter the following sequence:

            VK_SHIFTVK_DOWNVK_SHIFT

To send the Capital 'A'.
   
    VK_SHIFTAVK_SHIFT

To highlight the whole document via the keystrokes  control home - shift control end:

    VK_CONTROLVH_HOMEVK_SHIFTVK_ENDVK_SHIFTVK_CONTROL

Do not forget to issue a duplicate shift at a duplicate control to reset their states.
